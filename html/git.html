Git

  • git init - инициализация нового репозитория Git.
  • git clone <URL> - клонирование удаленного репозитория по указанному URL.
  • git add <файл> - добавление изменений в указанный файл в индекс.
  • git commit -m "сообщение" - фиксация изменений в репозитории с указанием сообщения.
  • git push - отправка локальных commits в удаленный репозиторий.
  • git pull - получение и слияние изменений из удаленного репозитория.
  • git status - отображение состояния рабочей директории и индекса.
  • git branch - просмотр, создание или удаление веток.
  • git checkout <ветка> - переключение на указанную ветку.
  • git merge <ветка> - слияние указанной ветки с текущей.
